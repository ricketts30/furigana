furigana

A small jQuery plugin for furigana and the <ruby> HTML element.

The project is hosted at https://github.com/ricketts30/furigana

The ruby Element

The basic <ruby> HTML element is used to provide additional meaning to a piece of inline text.

This is predominatly used to give a translation of a foreign script (e.g. 明日(Ashita)) or to signify the reading of a character where there are several valid options.

In this japanese word the second character is pronounced "ji" and means 'written character': 漢 (かん) 字 (じ)

however that character has another readings (and pronounciation) and so furigana are sometimes required to differentiate meanings (if it is not clearly obvious from context):

  • 大 (おお) 字 (あざ) pronounced "ooaza" and means 'large section of village'
  • 大 (おお) 字 (じ) pronounced "ooji" and means 'large written character'

This can also be used to unpack the meaning of an acronym like PRN(Pro Re Nata)

Demonstration
